静态元素
使用 jlang 和 jname 属性可以渲染静态的dom元素。请看一个例子：
index.js 或者 index.html 中插入以下代码：
Jet.lang.use(['cn','en']);
html模板中：
<div jlang> <span jname='cn'>你好，世界</span> <span jname='en'>hello world</span> </div> <button jon='click:Jet.lang.type="cn"'>设置中文</button> <button jon='click:Jet.lang.type="en"'>设置英文</button>
以下是输出的html:
你好，世界 hello world
每个jlang元素中只能包含n个jname元素，其中n是设置的语言的数量，Jet会根据当前选中的语言来自动替换掉jlang元素的内容。
