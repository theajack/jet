Jrun
使用Jrun，可以给html元素添加一个或者多个加载之后的回调函数,使用格式是 Jrun="function" 。 Jrun的值可以是一个Jet函数，全局函数或者是一串js代码。使用jet方法时可以使用多个，之间用逗号隔开。该回调函数this指代Jet元素，另有一个json参数，有三个属性：ele,data和jet， 分别指代绑定的html元素，绑定的数据和对应的jet元素。请看下面一个例子：
<div jrun='func'>单个Jet方法</div> <div jrun='func,func2'>多个Jet方法</div> <div jrun='func(this,opt)'>全局函数</div> <div jrun='this.$dom.js.text="js代码："+opt.ele.innerHTML'>js代码</div> <br> <div jdom='method'></div> <div jdom='method1'></div> <div jdom='method2'></div> <div jdom='function'></div> <div jdom='js'></div> <script> new Jet({ func:{ func:function(opt){ this.$dom.method.text=opt.ele.innerHTML+';'+ typeof opt.data+';'+typeof opt.jet; }, func1:function(opt){ this.$dom.method1.text='第一个jet方法：'+opt.ele.innerHTML+';'+ typeof opt.data+';'+typeof opt.jet; }, func2:function(opt){ this.$dom.method2.text='第二个jet方法:'+opt.ele.innerHTML+';'+ typeof opt.data+';'+typeof opt.jet; }, } }); function func(jet,opt){ jet.$dom.function.text='全局函数：'+opt.ele.innerHTML } </script>
