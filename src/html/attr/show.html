Jshow
1.使用布尔值
使用Jshow属性，可以根据一个布尔值或者布尔表达式来显示或者隐藏绑定的html元素。请看下面一个例子：
<input jon='click:this.bool=!this.bool' type="button" value='改变布尔值'> <div jshow='bool'> 示例html元素 </div> <script> new Jet({ data:{ bool:true } }); </script>

2.使用布尔表达式
jshow的条件使用一个布尔值表达式，与使用布尔值不同的是布尔表达式中的变量要使用 {{}} 包裹起来。请看一个例子
改变绑定数据的值： <input j='number' type="text"> <div jshow='{{$.number}}>10'> 示例html元素 </div> <script> new Jet({ data:{ number:10 } }); </script>
