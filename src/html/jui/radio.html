1.单选框
使用j-radio类来定义一个单选框。单选框可选的属性有 value 、checked 和 disabled。单选框不支持颜色与大小的改变。 未使用单选框组包裹的单选框会被分配到一个默认的全局单选框组中。
value属性是设置单选框的值，如果没有设置value，则radio元素的html内容会设置为它的value。值得注意的是，在同一个单选框组中的单选框不同能有相同的value值，所以最好是对每个单选框设置一个值，如果都默认为空值，则会报错。
checked属性是设置单选框是否被选中。
disabled属性是设置单选框是否被禁用。
jui-bind:radio一般都是在单选框组中使用绑定属性，若是单独在每个单选框上使用，则在同一个单选框组下的所有单选框只需要有一个绑定了数据源就行，否则会报错，且该数据是一个字符串型
没有包裹在单选框组中的radio元素不支持jui-change属性。
请看一些例子：
1.静态的单选框：
<div class='j-radio'></div> <div class='j-radio'>是否</div> <div class='j-radio' value='yes'></div> <div class='j-radio' disabled>好的</div>

2.使用$jui方法改变checked：
<div class='j-radio' jdom='radio1'>单选</div> <button jon='this.$jui("radio1").checked=true'>选中</button>

2.单选框组
使用j-radio-group类来定义一个单选框组。单选框可选的属性有 value 、checked 和 disabled。单选框不支持颜色与大小的改变。 未使用单选框组包裹的单选框会被分配到一个默认的全局单选框组中。
value属性是设置单选框的值，等价与直接在HTML中写值，只不过写在value属性里的值不会被显示出来。值得注意的是，在同一个单选框组中的单选框不同能有相同的value值，所以最好是对每个单选框设置一个值，如果都默认为空值，则会报错。
checked属性是设置单选框是否被选中。
disabled属性是设置单选框是否被禁用。
jui-bind:在使用了单选框组的时候，必须在单选框组中使用绑定属性，若是没有使用单选框组，则只能绑定其中一个单选框，且该数据是一个字符串型
单选框组支持jui-change。
请看一个例子：
<div class='j-radio-group' jui-bind='radioGroup' jui-change='change'> <div class='j-radio'>A</div> <div class='j-radio'>B</div> <div class='j-radio'>C</div> <div class='j-radio'>D</div> </div> <span j='radioGroup'>"radioGroup="+$</span> <script> new Jet({ data:{ radioGroup:'B' },func:{ change:function(opt){ JUI.msg.info(opt.value); } } }); </script>
