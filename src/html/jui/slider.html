滑动选择器
所有的可配置的属性如下：
<div class='j-slider' min='0' max='100' width='150' value='0' jui-change=''></div>
以上属性值皆为默认值
绑定的value值范围必须在min-max之间。若是不在这个范围之间，Jet会将值强制设置到这个区间之内。
slider可以使用JUI中的 大小、颜色和禁止属性
width支持各种css单位，若是不带单位默认是px
请看一个例子：





<div class='j-slider success xs' width='200' value='40' jui-bind='slider1' jui-change='cc'></div> <span j='slider1'></span> <br> <div class='j-slider warn s' min='20' width='300' value='40' jui-bind='slider2' jui-change='cc'></div> <span j='slider2'></span> <br> <div class='j-slider' min='30' max='200' value='40' jui-bind='slider3' jui-change='cc'></div> <span j='slider3'></span> <br> <div class='j-slider info l' width='100%' value='40' jui-bind='slider4' jui-change='cc'></div> <span j='slider4'></span> <br> <div class='j-slider danger xl' width='20em' value='40' jui-bind='slider5' jui-change='cc'></div> <span j='slider5'></span> <br> <div class='j-slider xl' disabled min='20' width='200' value='40'></div> <script> new Jet({ data:{ slider1:10, slider2:40, slider3:50, slider4:90, slider5:150, },func:{ cc:function(opt){ console.log(this,opt) } } }); </script>
