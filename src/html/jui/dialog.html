dialog 可以使用 xs s m l xl full 类名来使用不同尺寸的对话框，默认为m
JUI.dialog 中有以下以下几个方法：
1.isOpen():判断当前是否有打开的对话框
2.clear():关闭所有的对话框
3.removeAll():删除所有的对话框
请看一个例子：
<button jon='this.dshow=!this.dshow'>弹出对话框</button> <button jon='isOpen'>是否有打开的对话框</button> <button jon='clear'>关闭所有对话框</button> <button jon='this.dshow2=!this.dshow2'>不能移动和没有关闭按钮的对话框</button> <div class='j-dialog' dialog-title='对话框' jui-bind='dshow'> <input class='j-input' type="text"> <button class="j-btn">确认</button> </div> <div class='j-dialog' dialog-title='对话框2' jui-bind='dshow2' dialog-drag='false' dialog-close='false'> <input class='j-input' type="text"> <button class="j-btn">确认</button> <button class="j-btn" jon='this.dshow2=!this.dshow2'>关闭</button> </div> <script> new Jet({ data:{ dshow:false, dshow2:false },func:{ isOpen:function(){ JUI.msg(JUI.dialog.isOpen()); },clear:function(){ JUI.dialog.clear(); } } }); </script>
当dialog打开时，若其尺寸或位置发生了改变，可以使用dialog元素提供的reinitPos方法来将其重新放置在屏幕正中间。dialog元素可以通过绑定的DOM元素的$jui属性获得。
