自定义验证样式
使用 onOnePass，onOneFail 和 useDefaultStyle 结合可以定制验证成功或者失败的提示信息，请看一个简单的例子：
<style> .unpass{ color:#f44; font-size:12px; } .pass{ color:#4f4; font-size:12px; } </style> 数字：<input type="text" jvalid='number'> <span jshow='!$.pass' class='unpass'>*请输入数字</span> <span jshow='pass' jrun='opt.ele.style.display="none"' class='pass'>*验证通过</span> <script> new Jet({ onmounted:function(){ var _this=this; Jet.valid.useDefaultStyle=false; Jet.valid.onOnePass(function(ele,err){ _this.pass=true; }); Jet.valid.onOneFail(function(ele,err){ _this.pass=false; }); }, data:{ pass:true } }) </script>
以下是输出的html:
数字： *请输入数字 *验证通过
