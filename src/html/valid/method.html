Jet.valid
Jet.valid对象中含有很多可以定制的属性的，以下是可用的属性或者方法的列表：(点及下面的列表可以跳转到对应位置)
1.language (设置验证提示信息语言，默认值为 CHINESE，可选值为CHINESE和ENGLISH，不区分大小写)
2.useDefaultStyle (是否使用默认的错误提示样式，默认值为true)
3.useOnInput (是否使用oninput事件触发验证，默认值为false)
4.showInPlaceHolder (是否将格式要求使用 placeholder 显示，默认值为false)
5.useAlert (是否使用alert方法打印出验证失败信息，默认值为false)
6.useJUI (是否使用JUI的消息框打印出验证失败信息，默认值为false)
7.init() (初始化验证元素)
8.validate() (代码调用验证表单)
9.validInput() (代码调用验证单个输入元素)
10.addRegExp() (修改或者添加验证的正则表达式)
11.addValidText() (修改或者添加验证失败的提示文字)
12.onOnePass() (单个输入元素验证成功的回调函数)
13.onOneFail() (单个输入元素验证失败的回调函数)
14.addValid() (给输入框添加验证)
15.resetValid() (重置框验证样式)
16.clearValid() (清除验证)
1.language
修改 Jet.valid.language 可以设置验证提示信息语言，默认值为 CHINESE，可选值为 CHINESE 和 ENGLISH，不区分大小写。 请看一个例子：
<div>输入电话号码：<input type="text" jvalid='phone' jdom='lang_dom'></div> <button jon="click:Jet.valid.language='english';Jet.valid.validInput('lang_dom')">设置英文</button> <button jon="click:Jet.valid.language='chinese';Jet.valid.validInput('lang_dom')">设置中文</button> <script> new Jet();//为了初始化 jon 元素 </script>
以下是输出的html:
输入电话号码：
2.useDefaultStyle
修改 Jet.valid.useDefaultStyle 可以设置使用默认的错误提示样式，默认值为 true。请看一个例子：
<div>输入电话号码：<input type="text" jvalid='phone' jdom='default_dom'></div> <button jon="click:Jet.valid.useDefaultStyle=false;Jet.valid.validInput('default_dom')">禁用默认样式</button> <button jon="click:Jet.valid.useDefaultStyle=true;Jet.valid.validInput('default_dom')">使用默认样式</button> <script> new Jet();//为了初始化 jon 元素 </script>

3.useOnInput
修改 Jet.valid.useOnInput 可以设置是否使用oninput事件触发验证，默认值为 false，在开启了useOnInput模式下，useDefaultStyle将会变为false，无法设置其为true。请看一个例子：
<div>输入电话号码：<input type="text" jvalid='phone' jdom='input_dom'></div> <button jon="useoninput">启用oninput</button> <button jon="banoninput">禁用用oninput</button> <script> new Jet({ func:{ useoninput:function(){ Jet.valid.useOnInput=true; Jet.valid.useJUI=true; Jet.valid.validInput('input_dom'); },banoninput:function(){ Jet.valid.useOnInput=false; Jet.valid.useJUI=false; Jet.valid.validInput('input_dom'); } } }); </script>

输入电话号码：

7.init
使用 Jet.valid.init() 可以初始化验证元素，适用于使用脚本动态添加dom元素之后对其做初始化操作。请看一个例子： 该方法接受一个可选参数，可以是jdom字符串，jdom元素，id或dom元素，当没有传入参数时则对整个页面所有验证进行初始化。 如果参数含有 jvalid 属性，那么只对当前元素进行初始化，否则对该元素所有子元素含有jvalid且没有初始化的元素进行初始化，请看一个例子：
<div jdom='init_dom'></div> <button jon="click:addInput">添加元素</button> <button jon="click:initInput">初始化元素</button> <script> new Jet({ func:{ addInput:function(){ this.$dom.init_dom.html='输入电话号码：' }, initInput:function(){ Jet.valid.init('init_dom'); //参数可以使用 Jet.DOM 元素 或者 dom元素 //使用参数时是对指定元素的所有子验证元素进行初始化 //也可以不使用参数，表示对所有未初始化的验证元素进行初始化 } } }); </script>

12.onOnePass

不为空:<input type="text" jvalid="notnull"><br> 数字:<input type="text" jvalid="number"><br> <button jon="click:onOnePass">设置验证通过的回调函数</button> <script> new Jet({ func:{ onOnePass:function(opt){ Jet.valid.onOnePass(function(ele){ alert(ele.value); }); opt.ele.innerText='设置成功'; } } }); </script>

14.addValid

<div> <input type="text" jdom='add_dom'> </div> <button jon="Jet.valid.addValid('add_dom','phone')">添加验证</button> <script> new Jet(); </script>

15.resetValid
使用 Jet.valid.resetValid() 可以重置验证输入框的样式。 该方法接受一个可选参数，可以是jdom字符串，jdom元素，id或dom元素，当没有传入参数时则对整个页面所有验证进行重置。请看一个例子：
<div jdom='reset_form'> 请先输入以出发验证失败：<input type="text" jvalid='phone' jdom='reset_dom'> </div> <button jon="Jet.valid.resetValid('reset_dom')">参数输入框</button> <button jon="Jet.valid.resetValid('reset_form')">参数为表单</button> <button jon="Jet.valid.resetValid()">缺省参数</button> <script> new Jet();//为了初始化 jon 元素 </script>

16.clearValid
使用 Jet.valid.clearValid() 可以清除验证输入框的样式并且清除验证。 该方法接受一个可选参数，可以是jdom字符串，jdom元素，id或dom元素，当没有传入参数时则对整个页面所有验证进行清除。请看一个例子：
<div jdom='clear_form'> 请先输入以出发验证失败：<input type="text" jvalid='phone' jdom='clear_dom'> </div> <button jon="Jet.valid.clearValid('clear_dom')">参数输入框</button> <button jon="Jet.valid.clearValid('clear_form')">参数为表单</button> <button jon="Jet.valid.clearValid()">缺省参数</button> <button jon="Jet.valid.addValid('clear_dom','phone')">重新添加验证</button> <script> new Jet();//为了初始化 jon 元素 </script>
